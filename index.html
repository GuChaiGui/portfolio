LIBERTY COMMERCE - E-commerce (équipe de 2)

Objectif du projet :

L'objectif de ce projet est d'être capable d'utiliser les bases de Laravel afin de monter un site d'e-commerce de A à Z. Le theme choisi était Pokémon.


Ce projet est composé de 3 grandes parties :

1 - HTML/CSS : Réalisation mauqettes des pages d'authentification, un layout, un catalogue, une page produit, un panier et une page admin

2 - PHP : Reprendre les maquettes du site d'e-commerce et les dynamiser en utilisant le framework PHP Laravel.

3 - JAVASCRIPT, AJAX : Finalisation du site. Mise en place de consentement de cookies, intégration du Javascript, intégration d'AJAX.


Technologies :


LARAVEL - HTML - CSS - PHP - JAVASCRIPT

TROUVE TA ZONE - Concours Programmation Web (Equipe gagnante)

Objectif du projet :

L'objectif de ce concours est de savoir manipuler et intégrer un projet de e-commerce existant. Nous avons réaliser une interface responsive avec un design soigné.

Le but est de permettre aux utilisateurs d'avoir une vue d'ensemble sur les différents marchés et produits disponible et par zone

Le site permet donc au client (ne travaillant pas dans l'informatique) de gérer le panier, d'ajouter des produits dans un catalogue, de gérer la liste de ses marchés


Technologies :

Shopify - Le choix est stratégique, dans le but de répondre aux besoins du client, c'est une plateforme de e-commerce simple à manier et configurer, pratique a utliser,et optimisé

MY CLICKER GAME - Programmation Jeu Web

Objectif du projet :

L'objectif de ce projet est de savoir programmer un jeu avec du JavaScript, manipuler un DOM, ainsi que les Cookies dans un délai de temps très restraint (3 jours, équipe de 2).

Nous étions amenés à créer notre propre "clicker game".


Ce style de jeu qui se veut chronophage et addictif est vastement représenté dans les jeux web et mobile.

Nous avons choisi le thème des chats, il fallait donc pouvoir cliquer sur un bouton et pouvoir acheter des minions en boutique. Les minions pouvant donner un avantage qui est un nombre de clique par seconde, le joueur peut donc laisser tourner le jeu et obtenir des golds


Technologies :


HTML - CSS - JAVASCRIPT

MAKE TECH GREAT AGAIN - Concours Proof Of Concept (Equipe gagnante)

Objectif du projet :

Trouver une idée de projet innovante sur la thématique tech for goods.

La thématique de notre Tech for Good, par l’utilisation de la technologie, est de protéger les personnes vulnérables mais également de créer du lien dans un réseau ferroviaire souvent froid & anonyme.


Notre solution est donc de grouper les inconnus ayant le même itinéraire, les rendant moins vulnérables le temps du trajet, voire plus, et ainsi empêcher un potentiel agresseur, qui sera dissuadé de s'attaquer à un groupe.

Par le biais de notre application que nous avons nommée "Fleets", vous pouvez lors de votre trajet en Métro, RER ou Bus, chercher 10 minutes avant le départ de votre train, ou bien entre deux arrêts, rejoindre un Fleet qui partage ce trajet et ainsi vous sentir en sécurité au côté de personnes souhaitant se protéger des agressions.

De plus, si vous vous entendez bien avec votre groupe, vous pouvez approfondir cette relation sur des trajets récurrents avec des Fleets sauvegardés.
